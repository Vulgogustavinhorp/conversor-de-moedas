# Conversor de Moedas

Um aplicativo simples em Python com interface gráfica Tkinter que converte valores entre diferentes moedas usando a API pública [Frankfurter](https://www.frankfurter.app).

---

## Funcionalidades

- Conversão entre várias moedas
- Consulta em tempo real das taxas de câmbio pela API Frankfurter
- Interface gráfica simples e fácil de usar
